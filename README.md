# SMM-агентство

Веб-сайт SMM-агентства с услугами по продвижению в социальных сетях.

## Технологии
- HTML5, CSS3, JavaScript (ES6+)
- Google Fonts (Playfair Display SC)
- Font Awesome 6.4.0
- Яндекс.Карты API

## Страницы
- Главная — hero, слайдер, услуги
- Услуги — карточки тарифов
- Контакты — карта, контакты
- FAQ — аккордеон
- Кейсы — портфолио проектов
- Отзывы — отзывы клиентов, форма

## Установка
1. Клонируйте репозиторий
2. Откройте index.html в браузере
